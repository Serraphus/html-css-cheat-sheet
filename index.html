Input Type Example Description Code Snippet
text Attributes for all types of input tags should include name, type, id, and value. <input type="text" name="username" id="username" value="username">
password This is a textbox field that hides the typed characters. Using the required attribute will prevent users from submitting the form while this field is blank. <input type="password">
number Specific attributes include min, max, and step. <input type="number">
range This produces a slider, also with the properties min, max, and step. <input type="range" min="1" max="10" step="1">
checkbox For selecting more than one option. Notice here we have associated a label with each checkbox id, in the format
<label for="id-name">. 		<input type="checkbox">
radio Gender
 For mutually-exclusive options. Again, we associate a label to each option, allowing the user to click on the associated text to select the option, instead of having to click on the radiobutton itself. <input type="radio" name="gender" value="male/female">
select
This tag creates a drop-down list for the user to choose from.
list This allows the user to select from a pre-made list, or if no option suits, they can type their own answer.

The value of the list attribute should match the value of the corresponding datalist id.

A datalist is essentially an unordered list of options.

<input list="browsers">

<datalist id="browsers">
<option value="Safari">
<option value="Firefox">
<option value="Google Chrome">
</datalist>
submit A submit button!

The value atrribute should contain the text you wish to display on the button.		 <input type="submit" value="Submit">
